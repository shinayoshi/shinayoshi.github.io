GitHub Pagesでブログ始めました

はてなブログからGitHub Pagesに変更しました。変更した理由は以下のとおりです。正直なところは流行に乗ってみたかっただけですが。。。

  1. Markdownで記述できる
  2. 投稿をGit等でバージョン管理できる
  3. 独自ドメインの利用に費用がかからない

これまでの投稿はリセットされますが、いずれ最新の情報にアップデートした上で再投稿するかもしれません。今後ともよろしくお願いします。
